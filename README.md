# Mirrors China

## Setup

```sh
# install dependencies
pnpm install

# serve with hot reload at localhost:5173
pnpm serve

# build for production with minification
pnpm build
```

Docker:

```sh
docker-compose up -d
```

## Data source

- static/data.json
